## Overview

CashFlowr is a mobile-first finance management app built to make personal budgeting simple and intelligent. Users can log transactions, categorise spending, view trends, and receive AI-generated suggestions to improve their financial decisions. The app was designed, developed, tested, and released to production as a solo project.

## Features

- 📊 Track income and expenses with custom categories
- 📈 Visual spending trends and monthly summaries
- 💾 Persistent local data storage
- 🎨 Clean, intuitive mobile UI
- 🌐 Cross-platform support (Android, with iOS-ready codebase)
